# RBI Update Tracker Agent

## Setup Instructions
1. **Clone the repository**: Run the following command in your terminal:
   ```bash
   git clone https://github.com/Swatiitaly01/AA1-RBI-updates.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd AA1-RBI-updates
   ```

3. **Install dependencies**: Make sure you have the required packages installed. If using Python:
   ```bash
   pip install -r requirements.txt
   ```

## Usage Instructions
1. **To run the agent**: Execute the following command:
   ```bash
   python rbi_tracker_agent.py
   ```

2. **Configuration**: Update the `config.json` file to set up your required parameters.

3. **View logs**: Logs can be viewed in the `logs/` directory for troubleshooting or monitoring purposes.

## Additional Notes
- Ensure that your environment meets all the prerequisites listed in the documentation.
- For further assistance, refer to the documentation or contact the repository maintainer.
